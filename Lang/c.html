The C Programming Language

Description

C is a general-purpose programming language. It was created in the 1970s by Dennis Ritchie and remains very widely used and influential. It has found lasting use in operating systems code (especially in kernels), device drivers, and protocol stacks, but its use in application software has been decreasing. C is commonly used on computer architectures that range from the largest supercomputers to the smallest microcontrollers and embedded systems.

Uses

  • Embedded Systems

  • Operating Systems

  • Web Servers

  • Compilers
